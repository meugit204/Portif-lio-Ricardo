Projeto Lista de Compras

O Lista de Compras é uma pagina desenvolvida utilizando HTML, CSS, e Angular, projetado para facilitar a gestão de listas de compras. Atendendo às necessidades práticas do dia a dia, o aplicativo oferece uma abordagem intuitiva e personalizável para tornar a experiência de compras mais eficiente e organizada. Funcionalidades Atuais: Atualmente, o aplicativo permite aos usuários criar e editar listas de compras de forma dinâmica. A interface amigável possibilita a adição, remoção e modificação de itens na lista de maneira rápida e fácil.
